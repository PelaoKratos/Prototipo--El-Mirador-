Diseña un prototipo MVP completo de alta fidelidad para una aplicación web llamada "El Mirador", un sistema de gestión de gastos comunes para edificios residenciales.

Contexto del negocio

El sistema permite a la administración, propietarios y arrendatarios gestionar gastos comunes, pagos, deudas, recibos, reclamos e informes financieros de manera digital.

Actualmente la administración realiza gran parte de estos procesos manualmente, por lo que la aplicación debe transmitir confianza, transparencia, eficiencia y facilidad de uso.

Objetivo del MVP

Crear una plataforma moderna que permita:

Gestionar residentes y departamentos.
Registrar gastos comunes.
Registrar y consultar pagos.
Visualizar deudas pendientes.
Identificar morosos.
Generar recibos.
Gestionar reclamos y solicitudes.
Generar informes financieros.
Facilitar el acceso de arrendatarios mediante código único.
